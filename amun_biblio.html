AMUN Bibliography Formatter

This tool is intended to help format bibliographies for AMUN documents, most notably the Issues at AMUN guide. This is an automated system, so it is extremely capable of mistakes, but it should be accurate for a first-order approach. Of course, the rules set in the AMUN Style Guide are what ultimately need to be followed.

UN Document

This format is intended for official United Nations documents; in particular, those with an official document number (starts with A/RES/ or similar)

Conventions and Treaties

This format is intended for major conventions and treaties, for example the Treaty on the Non-Proliferation of Nuclear Weapons or the Convention on the Prevention and Punishment of Crimes against Internationally Protected Persons, including Diplomatic Agents. If you're not sure if a convention or treaty is "major" enough, don't hesitate to ask! (but it most likely is)
